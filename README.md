# Aurora Weather — UI Demo

Premium-looking Weather UI demo using HTML, CSS, and vanilla JavaScript. This project is a front-end-only mock demonstrating glassmorphism, soft neumorphism, dynamic gradients, and particle effects.

Files
- index.html — Main HTML structure
- styles.css — Styling (glassmorphism + neumorphism + responsive)
- app.js — Interactivity, mock data, search/autocomplete, particles, and animations

Run
Open `index.html` in your browser.

Notes
- Data is mocked for UI demonstration. Replace the `loadWeatherMock` function with a real API call (e.g., OpenWeatherMap) to fetch live data.
- Recent searches are stored in `localStorage`.
- The UI supports light/dark via system preference; toggle with the top-right button.

Suggestions
- Integrate a real weather API and map condition codes to richer visuals.
- Replace the simple canvas sparkline with a charting library for richer charts.
